Construisez votre propre barre latérale Conky avec XTrose Conky.
XTrose Conky peut gérer plusieurs résolutions d'affichage et est facile à installer.
La barre latérale est constituée de plusieurs Conkys et est facile à étendre.



Composants
- Date/heure
- processeur
- Mémoire vive
- réseau
- batterie
- système



Préparation

conky-all doit être installé
$ sudo apt-get install conky-all



Installation

Téléchargez ou clonez XTrose Conky depuis GitHub :
$ git clone https://github.com/xtrose/conky.git

Déplacez le répertoire .conky dans votre répertoire personnel

Rendez les fichiers suivants exécutables :
$ chmod + x ~ / .conky / xtrose / xtrose_start
$ chmod + x ~ / .conky / XTrose / XTrose_end
$ chmod + x ~ / .conky / xtrose / xtrose_UHD / xtrose_start
$ chmod + x ~ / .conky / xtrose / xtrose_FullHD / xtrose_start

Démarrez XTrose Conky depuis la console :
$ bash ~ / .conky / xtrose / xtrose_start
 


Amusez-vous à tester
