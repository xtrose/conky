xtrose Conky

https://github.com/xtrose/conky.git
Costruisci la tua barra laterale Conky con xtrose Conky.
xtrose Conky è in grado di gestire più risoluzioni di visualizzazione ed è facile da installare.
La barra laterale è composta da diversi Conky ed è facile da espandere.

Componenti

- Appuntamento
- processore
- Memoria ad accesso casuale
- Rete
- batteria
- sistema

Preparazione

conky-all deve essere installato
$ sudo apt-get install conky-all

Installazione

Scarica o clona Xtrose Conky da GitHub:
$ git clone https://github.com/xtrose/conky.git

Sposta la directory .conky nella tua home directory

Rendi eseguibili i seguenti file:
$ chmod +x ~/.conky/xtrose/xtrose_start

$ chmod +x ~/.conky/xtrose/xtrose_end
$ chmod +x ~/.conky/xtrose/xtrose_UHD/xtrose_start
$ chmod +x ~/.conky/xtrose/xtrose_FullHD/xtrose_start

Avvia xtrose Conky dalla console:
$ bash ~/.conky/xtrose/xtrose_start

Divertiti a testare
