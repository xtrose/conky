Создайте свою собственную боковую панель Conky с помощью xtrose Conky.
xtrose Conky может работать с несколькими разрешениями экрана и прост в установке.
Боковая панель состоит из нескольких Conkys и легко расширяется.



Составные части
- Дата / время
- процессор
- Оперативная память
- сеть
- аккумулятор
- система



Tребование

должен быть установлен conky-all
$ sudo apt-get install conky-all



Mонтаж

Загрузите или клонируйте xtrose Conky с GitHub:
$ git clone https://github.com/xtrose/conky.git

Переместите каталог .conky в свой домашний каталог

Сделайте следующие файлы исполняемыми:
$ chmod + x ~ / .conky / xtrose / xtrose_start
$ chmod + x ~ / .conky / xtrose / xtrose_end
$ chmod + x ~ / .conky / xtrose / xtrose_UHD / xtrose_start
$ chmod + x ~ / .conky / xtrose / xtrose_FullHD / xtrose_start

Запустите xtrose Conky из консоли:
$ Баш ~ / .conky / xtrose / xtrose_start
 


Получайте удовольствие от тестирования
